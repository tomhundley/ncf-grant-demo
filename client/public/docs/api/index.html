Ministry Grant Tracker API

GraphQL API for managing donor-advised funds and ministry grants.

Overview

This API provides endpoints for:

  • Ministries - Manage charitable organizations that receive grants
  • Donors - Manage individuals who establish giving funds
  • Giving Funds - Donor-advised fund accounts holding charitable assets
  • Grants - Financial distributions from funds to ministries

Grant Workflow

Grants follow a defined lifecycle:

PENDING → APPROVED → FUNDED
       ↘ REJECTED

Authentication

This demo API does not require authentication. In production, implement JWT or OAuth authentication.

Base URL

  • Development: http://localhost:4000
  • Production: https://ncf-demo.thomashundley.com/api/graphql
